Game Instructions

1. The grid consists of lights (white = ON, blue = OFF).

2. Click a cell to toggle it and its adjacent neighbors.

3. Turn all lights ON to win the game.

4. Each game starts with a random solvable configuration.

Good luck, and have fun!

Game Tutorial

The goal of Lights Out is to turn ON all the lights on the grid.

Clicking a light toggles its state and the state of its immediate neighbors.

If all lights are turned ON, you win!

Remember: A blue cell means the light is OFF, and a white cell means the light is ON.

Addendum

This game, Lights Out , developed by Supriya Singarapu, is an engaging project that leverages HTML, CSS, and JavaScript to create an interactive and enjoyable user experience. The primary challenge in developing this game was ensuring the initial board configuration was both random and solvable. To achieve this, the program applies a sequence of random toggles to an empty grid, ensuring that the resulting configuration maintains solvability. By doing so, players can enjoy a fair and unpredictable start to each game.

Dynamic styling played a crucial role in enhancing user engagement. The color transitions between ON (white) and OFF (blue) states are visually clear, ensuring users can easily distinguish between active and inactive cells. Event handling was another vital aspect, as clicking any cell toggles its state and that of its neighbors. This interactivity showcases the power of JavaScript's DOM manipulation and event listeners.

Additional features, such as tracking the number of moves, elapsed time, and displaying a congratulatory message upon winning, were implemented to make the game more rewarding and competitive. The personalized touch—addressing players by their names and celebrating their achievements—adds an element of delight and encouragement.

This project also provided valuable experience in debugging logical errors, particularly when handling random board initialization. Moreover, integrating a responsive and user-friendly design reinforced the importance of clear instructions and accessible interfaces.

Overall, the Lights Out game exemplifies the synergy of front-end web technologies to create an interactive, educational, and entertaining application. This project deepened my understanding of grid-based algorithms, state management, and user-centric design principles, preparing me for more advanced challenges in web development.
